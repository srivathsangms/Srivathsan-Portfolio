# 🚀 Srivathsan GMS Portfolio

A modern cinematic portfolio website built using **React + Framer Motion + TailwindCSS** to showcase my journey as an AI Engineer focused on building real-world intelligent systems.

The portfolio highlights projects in **Computer Vision, Machine Learning, Automation and Embedded AI**, along with experience, achievements and technical profiles.

---

## ✨ Features

- Snap-scroll presentation navigation  
- Fluid animated cursor interaction  
- Dark / Light mode ripple transition  
- Interactive project modals  
- Nested scroll intelligent sections  
- Performance optimised animation system  

---

## 🧠 Tech Stack

- React (Vite)
- Framer Motion
- Tailwind CSS
- Canvas Particle Animation
- Custom Navigation Engine
